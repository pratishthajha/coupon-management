# Coupon Management System

Backend REST API for an e-commerce coupon management system. Supports creating coupons with eligibility rules, selecting the best coupon for a given user and cart, applying coupons with usage tracking, and viewing analytics.

## Tech Stack
- Node.js
- Express.js
- In-memory storage

## How to Run
npm install
npm start

text

Server runs at `http://localhost:3000`.
